# Auth Gateway da São Paulo Urbanismo

# url base GET

Todos os usuários
http://localhost:5000/auth-gateway-spurb/0.0.1

  • method: GET

# :id GET

Um usuário
http://localhost:5000/auth-gateway-spurb/0.0.1/:id

  • method: GET

# cadastrar POST

Cria um usuário
http://localhost:5000/auth-gateway-spurb/0.0.1/cadastrar

  • method: POST
  • bodyParams:
{
	 "email": "usuario@teste.com",
	 "password":"123456",
	 "nrprodam": "e0545454" // opcional
}

# login POST

Cria autorização
http://localhost:5000/auth-gateway-spurb/0.0.1/login

  • method: POST
  • bodyParams:
{
	 "email": "usuario@teste.com",
	 "password":"123456"
}

# logout DELETE

Deleta autorização
http://localhost:5000/auth-gateway-spurb/0.0.1/logout

  • path: /logout
  • method: DELETE
  • headers:
	{
		"authorization": "api-res-token"
	}

# id DELETE

Deleta usuário

http://localhost:5000/auth-gateway-spurb/0.0.1/:id

  • method: DELETE
  • headers:
{
	"authorization": "api-res-token"
}
